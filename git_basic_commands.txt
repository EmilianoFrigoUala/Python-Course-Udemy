create a new repository in github
git init in the desired folder
create .gitignore if desired
git add .
git commit -m "first commit"
git remote add origin https://github.com/gitmaster88/(name of the repo).git
git push -u origin master
                
…or push an existing repository from the command line
git remote add origin https://github.com/gitmaster88/Python-Course-Udemy.git
git push -u origin master


…to clone a new repo in an empty folder
git clone https://github.com/gitmaster88/Python-Course-Udemy.git
